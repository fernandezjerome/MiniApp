# Mini_app

This app uses SASS, Vue, AJAX, and PHP to Fetch API from database. Navigating around in the app using Vue lifecycle hooks, components, and AJAX.

## To run from MAMP/WAMP

Open MAMP OR WAMP and create a database named db_mini import the tables from the includes folder to that data base Edit the main.js to fecth the data from your current data base. Make sure that the PHP connect file is formated correct for your system Place the folder into your www or htdocs.

Database name = db_mini
